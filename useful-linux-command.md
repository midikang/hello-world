vim command
========

i - change to insert mode.
a - add after current cursor.

[ESC] - change to command mode.
x - dlete the character in the current cursor.
:wq - save and quit 
:w - save
:q - quit
o - add one new line.
0 - go to line beginning.
$ - go to line end

yy - copy current line  
p - paste
u - undo
